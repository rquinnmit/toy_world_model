# Toy 3D Voxel World Model

A neural network that learns the dynamics of a 3D voxel world from first-person observations, then uses that learned understanding to imagine future states, play inside its own dreams, and reason about counterfactual outcomes.

## What is a World Model?

A world model is a neural network that learns to predict what happens next in an environment. Given the current observation and an action, it predicts the next observation and reward. Once trained, the model can "imagine" entire trajectories without the real environment -- enabling planning in imagination, curiosity-driven exploration, and interactive play inside a neural network.

## Architecture

The world model follows the DreamerV3 architecture (Hafner et al., 2023), simplified for a toy setting:

- **Encoder**: 4-layer CNN compressing 64x64 RGB frames into 256-dim latent vectors
- **RSSM**: Recurrent State Space Model with GRU memory (256-dim) and categorical stochastic latent (32 variables x 32 classes)
- **Decoder**: Transposed CNN reconstructing frames from the RSSM state
- **Reward Predictor**: MLP predicting scalar reward from state

The environment is a 3D voxel world (16x16x8) with first-person raycasting, procedurally generated rooms and corridors, hazards (lava), and goals.

## Setup

```bash
conda create -n toyworld python=3.11
conda activate toyworld
pip install -r requirements.txt
```

## Usage

### 1. Collect training data

```bash
python -m data.collector
```

Generates 200 episodes of exploration across procedurally generated levels. Data is saved to `data/episodes/`.

### 2. Train the world model

```bash
python train.py
```

Trains for 100 epochs. Checkpoints are saved to `checkpoints/` every 10 epochs. Monitor training:

```bash
tensorboard --logdir runs/
```

### 3. Visualize predictions

```bash
python -m viz.compare
```

Produces side-by-side comparisons of real trajectories vs. the model's imagined trajectories, with per-pixel error heatmaps.

### 4. Play inside the neural network

```bash
python play.py
```

Opens an interactive window where every frame is generated by the world model. WASD to move, arrow keys to turn, ESC to quit. No real environment is used -- the game exists entirely inside the neural network.

### 5. Curiosity-driven exploration (optional)

```bash
python explore.py --train   # Train an ensemble of 3 world models
python explore.py           # Run exploration driven by ensemble disagreement
```

Uses prediction uncertainty as intrinsic motivation. The agent seeks out areas where the ensemble of world models disagrees, actively exploring unfamiliar parts of the environment.

### 6. Counterfactual multiverse (optional)

```bash
python multiverse.py
```

From a given moment, branches all possible actions and visualizes the tree of alternate futures. Shows what would have happened with different choices.

## Innovations

**Neural Game Engine**: After training, the world model acts as a standalone game engine. The player provides keyboard input, and the neural network generates every frame from pure imagination. Inspired by GameNGen (Google, 2024).

**Uncertainty-Driven Exploration**: An ensemble of world models estimates epistemic uncertainty. High disagreement between models indicates unfamiliar territory. This uncertainty signal drives a curiosity policy that actively explores the unknown.

**Counterfactual Multiverse**: Recursive branching over all actions creates a tree of possible futures, enabling "what if" reasoning and visualization of alternate timelines from any decision point.
